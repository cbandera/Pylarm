# Pylarm

A small command line alarm utility written in Python

![An example of pylarm]


# Installation

If you don't use `pipsi`, you're missing out.
Here are [installation instructions](https://github.com/mitsuhiko/pipsi#readme).

Simply run:

    $ pipsi install .


# Usage

To use it:

    $ pylarm --help
